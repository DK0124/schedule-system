# 智能排班管理系統

一個現代化的純前端排班管理系統，具有 iOS 風格介面設計，支援 GitHub Pages 直接部署。

## 🚀 快速開始

### 線上使用
直接訪問：`https://[您的GitHub用戶名].github.io/schedule-system/`

### 本地使用
1. Clone 專案：
```bash
git clone https://github.com/[您的GitHub用戶名]/schedule-system.git
```
2. 直接開啟 `index.html`

## 📱 功能特色

- ✅ **員工管理**：支援正職/兼職員工分類
- 📅 **智能排班**：拖放式排班介面
- 💾 **自動儲存**：使用 localStorage 永久保存
- 📊 **資料匯出**：CSV/JSON 格式匯出
- 🎨 **現代化介面**：iOS 風格設計
- 📱 **響應式設計**：支援所有裝置
- 🔄 **即時同步**：支援多視窗同步
- 🌙 **深色模式**：自動適應系統主題

## 🛠️ 技術架構

- 純 HTML/CSS/JavaScript
- 無需後端伺服器
- 無需安裝任何套件
- 支援 GitHub Pages 部署

## 📖 使用說明

### 員工管理
1. 點擊「新增員工」按鈕
2. 填寫員工資料（姓名、類型、聯絡方式）
3. 設定可工作時段

### 排班操作
1. 點擊日曆上的日期
2. 選擇員工和班次
3. 可拖放調整排班

### 資料備份
- 自動儲存到瀏覽器
- 支援匯出 CSV/JSON
- 支援匯入備份檔案
